# **App Name**: AlphaScan AI

## Core Features:

- 数据采集与处理: 支持用户通过CSV文件上传行情数据或通过数据中心拉取，由Python引擎自动处理并生成多周期K线与技术指标数据，持久化存储。
- 交互式行情可视化: 集成TradingView Lightweight Charts或Apache ECharts展示K线图和多项技术指标，提供流畅的缩放、拖动和十字光标联动功能。
- 技术指标参数配置与查看: 提供对MA、BOLL、MACD等标准行业技术指标的参数调节功能，并在图表上实时展示，确保与主流软件精度一致。
- 回测条件构建面板: 设计用户友好的多指标、多周期筛选面板，支持固定字段顺序、比较符、数值输入，以及AND/OR链式逻辑组合构建复杂的条件策略。
- 概率统计回测引擎: 根据用户设定的指标条件，利用后端计算引擎统计历史行情中特定信号触发后，不同时间窗口内上涨/下跌的次数、概率、平均涨跌幅和样本数。
- 回测结果呈现: 可视化展示回测分析结果，包括统计表格、概率分布图等，清晰呈现策略的历史表现数据。
- PWA用户体验优化: 实现完整的PWA架构，包含manifest.json、Service Worker缓存策略，支持添加到主屏幕、HTTPS、后台状态恢复和离线UI加载提示。

## Style Guidelines:

- 深色方案。主色（功能交互、图表元素）：深蓝色 '#4D4FFF' (HSL: 240, 100%, 65%)，象征专业与深度。
- 背景色：极深蓝灰 '#17191C' (HSL: 240, 20%, 10%)，提供沉稳的阅读和工作环境。
- 强调色（高亮、关键信息）：明亮的青色 '#7AEBFF' (HSL: 187, 100%, 73%)，用于吸引注意力，提供对比度。
- 标题字体: 'Space Grotesk' (sans-serif)，赋予应用现代科技感和清晰度。
- 正文字体: 'Inter' (sans-serif)，提供在大量数据和图表内容下的极佳可读性和中立客观性。
- 采用一套简洁、线条分明的矢量图标，与专业量化分析工具的现代风格保持一致，确保在各种尺寸下的清晰度。
- 采用模块化的仪表盘布局，可根据用户偏好灵活组织行情图表、策略配置面板和回测结果区域，实现高效的信息概览和操作。
- 提供流畅的过渡动画，特别是在图表缩放、拖动和数据加载时，以增强用户体验的顺滑感和即时反馈，但不分散注意力。
